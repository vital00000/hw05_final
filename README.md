 Описание:
***
#### Yatube сайт социальная сеть на котором можно зарегистрировать пользователя. Вы можете комментировать посты пользователей. Можно также подписываться на разных авторов находящихся на сайте. Незарегестрированный пользователя могут только просматривать посты и оставлять коментарий
***

### **Пользовательские роли:**:
***
```
Аноним — может просматривать описания произведений, читать отзывы и комментарии.;

Аутентифицированный пользователь (user) — может читать всё, как и Аноним, дополнительно может оставлять комментарии. Может редактировать и удалять свои комментарии. Эта роль присваивается по умолчанию каждому новому пользователю.

Администратор (admin) — полные права на управление проектом и всем его содержимым.

Администратор Django — те же права, что и у роли Администратор.

```
### **Как запустить проект:**
```
- Клонировать репозиторий и перейти в него в командной строке:

  - git clone https://github.com/vital00000/hw05_final
  - cd hw05_final

- Cоздать и активировать виртуальное окружение:
    - python3 -m venv env
    - source env/bin/activate

- Установить зависимости из файла requirements.txt:
    - python3 -m pip install --upgrade pip
    - pip install -r requirements.txt
    
- Выполнить миграции:
    - python3 manage.py migrate
    
- Запустить проект:
    -python3 manage.py runserver
```
